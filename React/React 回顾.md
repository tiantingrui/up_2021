+ JSX

  JSX特点：

  + 不是模板语言，而是JS的语法糖
  + 在JS中直接写HTML标记
  + 本质用JS创建DOM节点，并可以 生成一个节点数
  + JSX的常用写法：表达式、延展、表达是作为子元素

  优点：

  + 允许使用熟悉的语法来定义 HTML 元素树；
  + 提供更加语义化且移动的标签；
  + 程序结构更容易被直观化；
  + 抽象了 React Element 的创建过程；
  + 可以随时掌控 HTML 标签以及生成这些标签的代码；
  + 是原生的 JavaScript

+ 以组件的方式来构建页面

  + 组件是构建页面的最小单元，可以组合
  + 组件的创建遵循单一职责原则，每个组件只做一件事
  + props和state构成了组件的view
  + react组件一般不提供方法，而是某种状态机
  + react组件可以理解成一个纯函数
  + 自上而下的单项数据绑定
